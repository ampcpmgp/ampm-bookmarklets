# ampm-bookmarklets

個人作成の Bookmarklet コードを管理・公開するためのプロジェクトです。

## 特徴

- 📚 ブックマークレットの一元管理
- 🌐 GitHub Pages で即利用可能
- 🚀 シンプルで使いやすいインターフェース
- 🔄 自動ビルド＆デプロイ

## 使い方

### 1. ブックマークレットページにアクセス

[**GitHub Pages**](https://ampcpmgp.github.io/ampm-bookmarklets/) にアクセスしてください。

### 2. ブックマークレットをブックマークバーに追加

使いたいブックマークレットのリンクをブックマークバーにドラッグ&ドロップします。

> **ヒント**: ブックマークバーが表示されていない場合は、`Ctrl+Shift+B` (Mac: `Cmd+Shift+B`) で表示できます。

### 3. 使いたいサイトで実行

ブックマークバーに追加したブックマークレットをクリックすると、現在開いているページで実行されます。

## 利用可能なブックマークレット

現在、以下のブックマークレットが利用可能です：

- **Copy Markdown Link** 📋 - 現在のページの Markdown リンクをコピー
- **Memo** 📝 - ページ上にメモを作成・管理
- **Translation Helper** 🌐 - 翻訳作業をサポート

最新のブックマークレット一覧は [GitHub Pages](https://ampcpmgp.github.io/ampm-bookmarklets/) でご確認ください。
